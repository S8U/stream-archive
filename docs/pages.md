# Stream Archive 페이지 구조

## 1. 인증 관련 페이지

### 1.1 로그인
- **경로**: `/login`
- **권한**: 비로그인 사용자
- **기능**:
  - 이메일/비밀번호 로그인
  - 로그인 실패 시 에러 메시지 표시
  - 로그인 성공 시 홈으로 리다이렉트

### 1.2 회원가입
- **경로**: `/signup`
- **권한**: 비로그인 사용자
- **기능**:
  - 사용자명, 이메일, 비밀번호 입력
  - 유효성 검사 (중복 체크, 형식 검사)
  - 회원가입 성공 시 로그인 페이지로 이동

## 2. 공개 페이지 (비로그인 접근 가능)

### 2.1 홈
- **경로**: `/`
- **권한**: 모두
- **기능**:
  - 공개(PUBLIC) 채널 목록 표시
  - 최근 녹화된 영상 목록
  - 채널별 카드 형태 또는 리스트 형태 표시

### 2.2 채널 상세
- **경로**: `/channels/[uuid]`
- **권한**: 채널 공개 범위에 따라 다름
  - PUBLIC: 모두 접근 가능
  - UNLISTED: 링크를 아는 사용자만
  - PRIVATE: 관리자만
- **기능**:
  - 채널 정보 표시 (이름, 프로필 이미지)
  - 연동된 플랫폼 목록
  - 녹화 스케줄 목록 (플랫폼별)
  - 해당 채널의 녹화 영상 목록 (최신순)
  - 녹화 중인 방송 표시
  - **관리자 전용 기능**:
    - 채널 편집 버튼 (모달 - `/admin/channels`의 채널 수정 모달 재활용)
    - 플랫폼 추가 버튼 (채널 편집 모달 내)
    - 플랫폼별 수정/삭제 (채널 편집 모달 내)
    - 스케줄 추가 버튼 (모달 - `/admin/schedules`의 스케줄 생성 모달 재활용)
    - 스케줄별 수정/삭제 (모달 - `/admin/schedules`의 스케줄 수정 모달 재활용)

### 2.3 비디오 시청 (실시간 시청 포함)
- **경로**: `/videos/[uuid]`
- **권한**: 비디오 공개 범위에 따라 다름
- **기능**:
  - HLS 비디오 플레이어
    - 녹화 완료된 영상 재생
    - 녹화 중인 방송 실시간 시청 (약 3초 딜레이)
  - 실시간 채팅 리플레이 (영상과 동기화)
  - 시청자 수 그래프 (완료된 영상) / 현재 시청자 수 (녹화 중)
  - 방송 제목 변경 이력 표시
  - 카테고리 변경 이력 표시
  - 영상 정보 (제목, 길이, 파일 크기, 녹화 시간)
  - 녹화 상태 표시 (녹화 중 / 완료)

## 3. 관리자 페이지

### 3.1 관리자 대시보드
- **경로**: `/admin`
- **권한**: ADMIN
- **기능**:
  - 현재 녹화 중인 방송 목록
  - 시스템 상태 (CPU, 메모리, 디스크 사용량)
  - 최대 동시 녹화 수 현황
  - 최근 녹화 실패 목록

### 3.2 채널 관리
- **경로**: `/admin/channels`
- **권한**: ADMIN
- **기능**:
  - 전체 채널 목록 표시
  - 채널 검색 및 필터링
  - 채널 생성 버튼 (모달)
  - 채널별 액션 (수정 모달, 삭제)

**모달: 채널 생성/수정**
- 채널 이름 입력
- 공개 범위 설정 (PUBLIC/UNLISTED/PRIVATE)
- 플랫폼 연동 관리
  - 플랫폼 추가 버튼
    - 플랫폼 선택 (치지직, 트위치, 아프리카TV, 유튜브)
    - 플랫폼 채널 ID 입력
    - 프로필 동기화 활성화 여부
  - 연동된 플랫폼 목록 (수정 시)
    - 플랫폼별 수정 (채널 ID, 프로필 동기화 설정)
    - 플랫폼별 삭제
- 채널별 설정 관리 (수정 시)

### 3.3 녹화 스케줄 관리
- **경로**: `/admin/schedules`
- **권한**: ADMIN
- **기능**:
  - 전체 녹화 스케줄 목록
  - 채널별, 플랫폼별 필터링
  - 스케줄 유형별 필터링
  - 스케줄 생성 버튼 (모달)
  - 스케줄별 액션 (수정 모달, 삭제)

**모달: 스케줄 생성/수정**
- 채널 선택
- 플랫폼 선택
- 스케줄 유형 선택
  - 한 번만 (ONCE)
  - 매번 (ALWAYS)
  - 매주 n요일 (N_DAYS_OF_EVERY_WEEK) - 요일 다중 선택
  - 날짜 지정 (SPECIFIC_DAY) - 날짜 다중 선택
- 녹화 품질 설정

### 3.4 녹화 기록 관리
- **경로**: `/admin/records`
- **권한**: ADMIN
- **기능**:
  - 전체 녹화 기록 목록 (진행 중, 완료, 취소)
  - 상태별 필터링 (녹화 중, 완료, 취소)
  - 채널별, 플랫폼별 필터링
  - 녹화 중인 방송 강제 종료
  - 녹화 상세 보기 (모달)

**모달: 녹화 상세**
- 녹화 상세 정보
- 녹화 로그 표시
- 연관된 비디오 링크
- 녹화 취소 기능

### 3.5 비디오 관리
- **경로**: `/admin/videos`
- **권한**: ADMIN
- **기능**:
  - 전체 비디오 목록
  - 채널별, 공개 범위별 필터링
  - 검색 (제목)
  - 비디오 편집 버튼 (모달)
  - 비디오 삭제

**모달: 비디오 편집**
- 제목 수정
- 공개 범위 변경 (PUBLIC/UNLISTED/PRIVATE)

### 3.6 시스템 설정
- **경로**: `/admin/settings`
- **권한**: ADMIN
- **기능**:
  - 글로벌 설정 섹션
    - 기본 보존 기간 설정 (일)
    - 최대 동시 녹화 수 설정
    - API 폴링 주기 설정 (초)
    - 기본 녹화 품질 설정
  - 설정값 인라인 수정

### 3.7 사용자 관리
- **경로**: `/admin/users`
- **권한**: ADMIN
- **기능**:
  - 전체 사용자 목록
  - 사용자 검색 (이름, 이메일)
  - 역할 변경 (ADMIN ↔ USER)
  - 사용자 삭제

## 4. 사용자 페이지

### 4.1 시청 기록
- **경로**: `/history`
- **권한**: 로그인 사용자
- **기능**:
  - 시청한 영상 목록 (시청 시간 역순)
  - 각 영상별 정보
    - 썸네일
    - 제목
    - 채널명
    - 시청 시각
    - 재생 위치 (진행률 바)
  - 영상 클릭 시 이어보기 (마지막 재생 위치부터)
  - 시청 기록 개별 삭제
  - 전체 시청 기록 삭제

### 4.2 마이페이지
- **경로**: `/my`
- **권한**: 로그인 사용자
- **기능**:
  - 프로필 정보 표시
  - 비밀번호 변경
  - 이메일 변경

## 5. 에러 페이지

### 5.1 404 Not Found
- **경로**: `/404`
- **기능**: 존재하지 않는 페이지 안내

### 5.2 403 Forbidden
- **경로**: `/403`
- **기능**: 접근 권한 없음 안내

### 5.3 500 Server Error
- **경로**: `/500`
- **기능**: 서버 오류 안내
